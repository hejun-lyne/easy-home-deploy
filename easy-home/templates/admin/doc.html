com.h2c.app

主要是Token安全访问机制，云端分配AppKey及AppSecret到App(请求端)，App使用AppKey及AppSecret刷新Token以正常访问Api。

使用Token调用Api

注意这里的所有api都必须带上请求参数token，在下面的接口说明中不再列举，如果token非法会返回error=100错误；

URL: /app/api

com.h2c.app.refreshToken

刷新Token，Token默认有效期为600秒，超过有效期的token无法用于调用api，需要再次刷新之后方可使用。
请求参数 name com.h2c.app.refreshToken required 接口名称
v 1 required 接口版本
appId 41los2nfo1209af29slf9fsd required AppId(必须使用Base64编码)
encrypt true | false required 是否使用加密(Base64编码之后进行AES加密, appSecret作为key)
返回结果 success

{
        name: com.h2c.app.refreshToken,
        v: 1,
        error: 0,
        token: 8jsdf23alp1p209afljlfwsdx2,
        expires_in: 600
}

返回新的Token，同时包括有效期
error_-1

{
        name: com.h2c.app.refreshToken,
        v: 1,
        error: -1,
        err_msg: AppId解密失败：Exception
}

解密AppId出现异常，请检查加密算法是否有误
error_1

{
        name: com.h2c.app.refreshToken,
        v: 1,
        error: 1,
        err_msg: AppId无效
}

如果是非法的AppId则返回该错误

com.h2c.homekit

智能设备云端服务接口，提供命令获取、状态更新等功能。

URL: /homekit/api

com.h2c.homekit.getCommand   (GET)

获取智能设备的执行指令，该接口应该由智能设备调用。
请求参数 name com.h2c.homekit.getCommand required 接口名称
v 1 required 接口版本
token io9823jks87sdfn2sdfadf4erdfsdadf23es required Token（com.h2c.app.refreshToken）
mac 11:22:33:44:55:66 required 智能设备物理地址
type AirBox | IrBox required 智能设备类型，目前只有两种
identifier 空气盒子 | 智能遥控 | ... required 智能设备名称
firm 2.0.6 | ... optional 当前固件版本
acl {admins:[1], users:[2,3]} optional 访问控制列表内容，使用JSON格式
state {temperature:26, humidity:86, pm25:220, illuminance:23, voc:12.8} optional 设备状态数据，使用JSON格式
result 0(success) | ERR_CODE optional 上次命令执行结果
返回结果 success_1 0x00... 返回指令数据，以0x00引导
success_2 0xF0 请求超时（无指令需要执行）
error_1 0xE0 Token无效，可能是非法Token或Token已过期
error_2 0xE1 Indentifier解析失败，请使用UTF-8编码，通过hex传输
error_3 0xE2 Acl解析失败，请使用JSONObject格式
error_4 0xE3 State解析失败，请使用JSONObject格式
